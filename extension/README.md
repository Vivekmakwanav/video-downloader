# Vidnexa Video Downloader Chrome Extension

This browser extension adds a one-click **Download** button directly to YouTube watch pages. When clicked, it routes the video URL directly to your deployed Vidnexa production server.

## Installation Instructions

1. Open your Google Chrome (or Chromium-based) browser.
2. Navigate to **`chrome://extensions/`**.
3. Toggle the **"Developer mode"** switch in the top-right corner to **ON**.
4. Click the **"Load unpacked"** button in the top-left corner.
5. Select the **`extension/`** folder inside your project repository.
6. The extension is now active! Go to any YouTube video page, and you will see a colorful **Download** button right next to the channel subscribe button.
